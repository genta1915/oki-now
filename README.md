# OKI NOW

沖縄で「今日どこ行こう？」を解決する、遊び場所提案Webサイトです。

## 公開URL

🌐 ※ 現在開発中

## GitHub

🚀 https://github.com/genta1915/oki-now

## 概要

OKI NOWは、沖縄で遊びたい人向けに、条件に合ったスポットを提案するWebサイトです。

「誰と遊ぶか」「気分」「目的」などから、沖縄のおすすめスポットを探せます。

## 主な機能

- 誰と遊ぶかを選択
- 気分や目的に合わせたスポット提案
- スポット画像の表示
- おすすめ理由の表示
- Googleマップへのリンク

## 制作目的

沖縄には魅力的な場所が多い一方で、「今日どこ行く？」と迷うことも多いです。

OKI NOWは、沖縄の若者や観光客が、今の気分に合った遊び場所を簡単に見つけられるサービスを目指して制作しています。
